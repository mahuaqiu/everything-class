核心功能

  • 实时搜索 - 输入即刻过滤，支持 PID / 进程名称 / 标题 / Class Name
  • 定位模式 - 点击"定位"按钮，鼠标变成瞄准状态，点击目标窗口获取信息
  • 层级切换 - 默认顶层窗口，点击"树形"切换查看子窗口层级
  • 进程筛选 - 输入进程名称（如 notepad）列出该进程所有窗口

技术选型

  • 语言 - Rust（高性能、内存安全）
  • GUI框架 - native-windows-gui（原生Windows控件，体积~1MB）
  • Windows API - EnumWindows / GetClassName / GetWindowText / GetWindowThreadProcessId

目标指标

  • 体积 < 2MB
  • 启动时间 < 100ms
  • 搜索响应 < 50ms（千条数据）
